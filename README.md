# Locus-of-Memory
3D表現Ⅱのチーム制作

Gitクローンを作るときの手順
・複製したいプロジェクトのURLをコピー（GitページのCodeから）する
・適当な場所（フォルダの中など）で右クリックし、Gitクローン(複製)を選択
・コピーしたURLを張り付けてOK

更新の手順
・クローンフォルダ(上記で作ったもの)の中で右クリックしてGit同期を選択
・コミットを選択して変更した内容にチェックをつける(メッセージに変更内容を記載するとわかりやすい)
・プルを選択して競合がないかチェック
・プッシュを選択して変更を確定

・何か更新した際の基本的な流れはコミット->プル->プッシュ

・どんな変更を行ったかをメッセージに書いてもらえると助かります

・作業開始前にほかの更新との競合を避けるため最初にプルを行うことを推奨

・同じ個所(model.txtの行など)を同時にいじるのは競合の危険があるため、事前に作業内容の報告をお願いします
